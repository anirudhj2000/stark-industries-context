### 2. Cloudflare API[^1] Surface Optimization

**Key Points:**
- Cloudflare API has approximately 2,600 endpoints
- Exposing each endpoint as a tool would require ~1.2 million tokens per call
- Creating a traditional MCP server for the entire API surface is impractical

**Solution Presented:**
- Matt Carry's approach: Expose only **two calls** - `Search` and `Execute`
- Both endpoints accept code as input (string of code)
- Search input receives the entire OpenAPI JSON spec
- Execute provides callable functions against discovered endpoints

**Result:**
- Token reduction: 1.2M+ tokens → ~1,000 tokens
- **99.9% reduction** in token usage
- Dramatically faster API interactions
